About

Hello, I'm Bruno Castanheira, a Game Developer, and also a gamer from Portugal. I started playing video games when I was a child, since then it's been my hobby. So when I started programming and got into making games, combining the two is something that I enjoy.

Also I have a bachelor degree in Digital Games Development Engineering

I have experience with C#, C++, Java, Unity, Unigine Engine. C# it's the one I used it most, for Unity, and Mono Game.

Java was the first language, in that time I was playing Minecraft and I got interested in creating plugins for it, that's why Java was my first programming language.

C# it's been my top language, it's the one that I'm more comfortable with.

On the C++ side, I started learning it at the University for a 3D Graphics Application. Recently I saw the Unigine Engine witch has support for C# and C++, since I already have more experience with C#, C++ was the way to go.
